If the Problem Persists After a Clean Install

If a clean installation does not resolve the issue, it may be hardware-related. Follow the steps below to troubleshoot your system.

Hardware Troubleshooting Checklist

Reseat System Memory (RAM)

  • Power down the system and unplug it.
  • Remove the RAM modules and reinstall them securely in their slots.

Verify Connections

  • Ensure all power and data cables are properly connected.
  • Check the motherboard, storage drives, and other peripheral connections.

Reseat Graphics Card

  • Remove the GPU and reinstall it in its PCIe slot.
  • Make sure the PCIe power connectors are fully seated.

Clear CMOS

  • Reset BIOS settings by clearing the CMOS battery.
  • Refer to the motherboard manual for exact steps.

Check Protective Stickers/Covers

  • Remove any stickers or protective covers on components, including the GPU PCIe connector.

Inspect Thermal Paste

  • Check the CPU and GPU for degraded or improperly applied thermal paste.
  • Reapply high-quality thermal paste if necessary to prevent overheating.

Monitor System Temperatures

  • Use software tools to check CPU and GPU temperatures under load.
  • Ensure fans and cooling solutions are functioning correctly.
